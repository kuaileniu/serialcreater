# 前言
- 涉及系统中的字符编码，全部采用UTF-8字符集。
- 关系数据库采用mysql5.7
- 缓存采用redis3.0
- JDK采用jdk8
- 需为机器预装MySQL、Redis、JDK。

# 系统初始化
## 创建关系数据库表接口
```sql
create table t_org_product
(
   id                   bigint not null auto_increment comment 'id',
   org_code             varchar(10) comment '组织编号',
   org_id               int comment '组织id',
   product_code         varchar(10) comment '产品编号',
   product_id           int comment '产品id',
   last_source_id       int default -1 comment '最后资源号',
   record_status        tinyint default -1 comment '此行记录状态：-1，未满仍可用；2已满，停用',
   record_owner         varchar(44) comment '所有者',
   record_owner_expire  datetime comment '所有者期满时间',
   create_time          datetime comment '创建时间',
   stop_time            datetime comment '停用时间',
   modify_time          datetime comment '修改时间',
   primary key (id)
)
ENGINE = MYISAM;
alter table t_org_product comment '组织产品组合表';

create table t_org_product_his
(
   id                   bigint not null auto_increment comment 'id',
   org_code             varchar(10) comment '组织类别编号',
   org_id               int comment '组织类别id',
   product_code         varchar(10) comment '产品类别编号',
   product_id           int comment '产品类别id',
   last_source_id       int default -1 comment '最后资源号',
   record_status        tinyint default -1 comment '此行记录状态：-1，未满仍可用；2已满，停用',
   record_owner         varchar(44) comment '所有者',
   record_owner_expire  datetime comment '所有者期满时间',
   create_time          datetime comment '创建时间',
   stop_time            datetime comment '停用时间',
   modify_time          datetime comment '修改时间',
   move_time            datetime comment '转移时间',
   primary key (id)
)
ENGINE = MYISAM;
alter table t_org_product_his comment '组织产品组合历史表';

create table t_zhs_lock
(
   id                   bigint not null comment 'id',
   service_id           varchar(100) comment '业务id',
   record_owner         varchar(44) comment '所有者',
   record_owner_expire  datetime comment '所有者期满时间',
   create_time          datetime comment '创建时间',
   modify_time          datetime comment '修改时间',
   primary key (id)
)
ENGINE = MYISAM;
alter table t_zhs_lock comment '锁表';
```

# 部署说明
{project}/deploy下配置不同环境配置和启动脚本，相应环境下的application.yml为系统配置文件，*.sh(*.md)为启动/关闭脚本。


## 修改mysql配置
修改相应环境下的application.yml内的片段
```
    url: jdbc:mysql://127.0.0.1:3306/serialcreator?useUnicode=true&characterEncoding=UTF-8&useSSL=false&createDatabaseIfNotExist=true
    username: root
    password:
```


## 修改Redis配置
修改相应环境下的application.yml内的片段
```
  redis:
     database: 0
     host: 127.0.0.1
     port: 6379
     password:
```

## 启动端口设置
### 默认设置
修改相应环境下的application.yml内的片段
```
server:
  port: 80
```  
### 脚本设置
启动脚本中添加 --server.port=80

例如 java -server -Xms256M -Xmx2048M -Xss2M -jar SerialCreator.jar --server.port=80


## 打包说明

打包不同环境时选择{project}下cmd脚本文件。

- 开发环境打包 package-dev.cmd
- 测试环境打包 package-dev.cmd
- 生产环境打包 package-dev.cmd
